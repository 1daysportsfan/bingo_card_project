🎲 棒球賓果卡 使用說明 / Baseball Bingo Help Guide

📌 遊戲目的 / Game Objective

在觀賞棒球比賽的同時，透過賓果卡遊戲提升趣味性。當比賽中發生卡片上的事件時，就可以點選對應格子。完成橫列、直行或對角線，即可達成 BINGO！

Enhance the baseball viewing experience with a fun bingo game. When an event listed on the card happens during the game, tap the corresponding square. Complete a row, column, or diagonal to get BINGO!

🔧 基本操作 / How to Play

  • 點擊任一格子以標記該事件（再次點擊可取消標記）。
    Tap any square to mark the event (tap again to unmark).
  • 完成任一橫列、直行或對角線後，格子上下方會顯示「BINGO!!」動畫提示。
    When you complete a row, column, or diagonal, an animated "BINGO!!" message will appear above and below the squares.
  • 按下「重新產生」按鈕可重新隨機產生一組新的賓果卡。
    Tap "Regenerate" to get a new random card.
  • 中間的「Free Space」代表免費格，可自由視為已達成。
    The "Free Space" in the center is always considered marked.

🌐 語言切換 / Language Switch

  • 可於右側選單中選擇「Chinese」、「English」或「Chinese-English」顯示事件名稱。
    Use the dropdown to choose "Chinese", "English", or "Chinese-English" for event names.

🖨️ 列印與下載 / Print & Download

  • 列印賓果卡： 直接列印目前畫面內容。
    Print Card: Print the current bingo card directly.
  • 列印說明： 系統會自動隱藏按鈕與工具列，只保留賓果卡進行列印。
    Print Mode: The system hides all buttons and toolbars, keeping only the card visible.
  • 下載圖片： 將整張賓果卡另存為 PNG 檔案。
    Download Image: Save the bingo card as a PNG file.
  • 批次匯出 PDF： 自訂張數後按下按鈕，系統會自動產生多張 PDF 格式賓果卡，適合多人活動使用。
    Export PDF: Specify how many cards you want and generate a multi-page PDF.

📱 手機使用說明 / Mobile Use

  • 畫面會自動切換為垂直排列：賓果卡在上方，按鈕在下方。
    The layout automatically adjusts for smaller screens with the bingo card on top and buttons below.
  • 賓果格會自動放大，方便點選。
    Squares enlarge for easier tapping.
  • 支援完整點擊、動畫效果、列印與下載功能。
    Fully supports all features like clicking, animation, printing, and download.
